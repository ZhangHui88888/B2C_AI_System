# 部署文档目录

本目录包含系统部署和配置相关的所有文档。

```
deployment/
├── 部署指南.md           # 【主文档】完整的部署流程和配置指南

```

## 📖 阅读顺序

### 新手入门（首次部署）

1. **先读**: [`部署指南.md`](./部署指南.md)
   - 这是主文档，包含完整的部署流程
   - 涵盖：系统架构、部署步骤、第三方服务配置、故障排除
   - 已经包含了环境变量的配置说明

2. **参考**: [`环境变量配置.md`](./环境变量配置.md)
   - 这是详细参考文档
   - 当你需要查看某个环境变量的详细说明时使用
   - 包含：每个变量的获取方式、示例值、安全注意事项

### 快速查询

- **我要部署系统** → 看 [`部署指南.md`](./部署指南.md)
- **我要查某个环境变量** → 看 [`环境变量配置.md`](./环境变量配置.md)
- **我遇到部署问题** → 看 [`部署指南.md`](./部署指南.md) 的"故障排除"章节

## 🎯 文档定位

### 部署指南.md（主文档）
- **目标读者**: 需要部署系统的开发者/运维人员
- **内容范围**: 从零开始到系统上线的完整流程
- **特点**: 
  - 步骤清晰，按顺序执行
  - 包含所有必要的配置
  - 提供故障排除方案
  - 包含生产环境检查清单

### 环境变量配置.md（参考文档）
- **目标读者**: 需要详细了解每个环境变量的开发者
- **内容范围**: 所有环境变量的详细说明
- **特点**:
  - 每个变量都有详细解释
  - 包含获取方式和示例
  - 说明安全注意事项
  - 提供常见问题解答

---

## 💡 使用建议

1. **首次部署**: 完整阅读 `部署指南.md`，按步骤操作
2. **配置环境变量**: 参考 `部署指南.md` 中的配置章节，需要详细说明时查看 `环境变量配置.md`
3. **遇到问题**: 先查看 `部署指南.md` 的故障排除章节
4. **日常维护**: 将 `环境变量配置.md` 作为快速参考手册
