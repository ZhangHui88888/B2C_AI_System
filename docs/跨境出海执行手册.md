## 1.2 测品选品

### 1.2.1 选品流程

```
市场调研 → 初选品类 → 平台测品 → 双轮选品 → 确定主推款
   ↓           ↓           ↓           ↓           ↓
 数据分析    5-10个品    小批量测试   数据验证    3-5个SKU
```

### 1.2.2 选品维度评估表

| 维度 | 评估标准 | 权重 | 得分(1-5) |
|-----|---------|-----|----------|
| **市场需求** | 搜索量、趋势、季节性 | 25% | |
| **竞争程度** | 竞品数量、头部垄断度 | 20% | |
| **利润空间** | 售价-成本-运费-广告 > 30% | 20% | |
| **供应链** | 货源稳定、起订量合理、品控可控 | 15% | |
| **物流友好** | 重量轻、体积小、无特殊限制 | 10% | |
| **合规风险** | 无专利侵权、认证简单 | 10% | |

### 1.2.3 测品方法

| 方法 | 做法 | 成本 |
|-----|------|-----|
| **广告测品** | Facebook/TikTok 投放测试广告 | $50-200/品 |
| **市场容量测试** | 用工具查询关键词搜索量 | 工具订阅费 |
| **海外认证** | 提前了解目标市场认证要求 | 视品类 |

### 1.2.4 选品工具推荐

| 工具 | 用途 | 价格 |
|-----|------|-----|
| Jungle Scout | 亚马逊选品、销量估算 | $49/月起 |
| Helium 10 | 关键词研究、竞品分析 | $39/月起 |
| Google Trends | 趋势分析、季节性判断 | 免费 |
| 1688/阿里巴巴 | 供应商寻源、成本估算 | 免费 |

## 1.3 制定战略

### 1.3.1 营销战略

| 要素 | 需要确定的内容 |
|-----|--------------|
| **目标市场** | 首选国家 1-2 个（建议美国或欧洲单一国） |
| **渠道策略** | 🟢 独立站优先 → 🟡 后期亚马逊 |
| **定价策略** | 高端/中端/性价比，对标竞品 |
| **推广策略** | 付费广告为主 / 网红为主 / SEO为主 |

### 1.3.2 出海路径规划

```
🟢 路径 A：独立站优先型（你的当前路径）
├── 阶段1：独立站 + 付费广告测试（0-3月）
├── 阶段2：网红合作 + 内容种草（3-6月）
└── 阶段3：SEO + 私域复购（6月+）

🟡 路径 B：亚马逊起步型（后期可选）
├── 阶段1：亚马逊开店，测试产品（0-6月）
├── 阶段2：稳定出单后建独立站（6-12月）
└── 阶段3：多渠道扩展（12月+）

🔴 路径 C：全渠道型（资金充足时）
├── 独立站 + 亚马逊 + TikTok Shop 同步
├── 海外仓备货
└── 线下渠道分销
```

### 1.3.3 供应链 & 运营 & 物流

| 环节 | 🟢 当前方案 | 🟡 后期升级 |
|-----|-----------|-----------|
| **货源** | 一件代发 / 小批量采购 | 自有工厂 / 大批量代工 |
| **生产周期** | 供应商现货 | 预下单+备货 |
| **物流** | 工厂直发（7-15天） | 海外仓（3-5天） |
| **库存** | 零库存 / 少量 | 海外仓备货 |

### 1.3.4 团队/代运营

| 模式 | 优点 | 缺点 | 阶段 |
|-----|------|-----|-----|
| **1人运营** | 成本最低 | 精力有限 | 🟢 当前 |
| **外包设计** | 专业度高 | 沟通成本 | 🟢 当前 |
| **自建团队** | 可控、积累经验 | 人力成本高 | 🔴 后期 |
| **代运营** | 快速启动、专业 | 成本高、依赖外部 | 🟡 中期 |

### 1.3.5 资金预算

| 项目 | 🟢 低成本启动 | 🟡 中等投入 | 🔴 规模化 |
|-----|-------------|-----------|---------|
| 首批备货 | $500-2000 | $5000-10000 | $20000+ |
| 广告测试 | $500-1000/月 | $3000-5000/月 | $10000+/月 |
| 建站费用 | $0（自有系统） | $0 | $0 |
| 工具订阅 | $50-100/月 | $200-300/月 | $500+/月 |
| 认证合规 | $0-500 | $1000-2000 | $3000+ |
| **总计（启动）** | **$3000-5000** | **$15000-30000** | **$50000+** |

## 2.1 亚马逊开店 🟡【中期-有资金时】

> ⚠️ 成本较高（月租 $39.99 + 佣金 15%），建议独立站稳定后再考虑

### 2.1.1 开店流程

| 步骤 | 具体操作 | 所需材料 |
|-----|---------|---------|
| 开通账户 | 注册亚马逊卖家账户 | 营业执照、法人身份证、双币信用卡 |

| 店铺运营 | 优化 listing、处理订单 | - |
| 激活广告营销 | 开通 PPC 广告 | 广告预算 |

### 2.1.2 亚马逊 vs 独立站对比

| 维度 | 亚马逊 | 独立站（你的 B2C 系统） |
|-----|-------|---------------------|
| 流量 | 平台自带 | 需要自己引流 |
| 信任度 | 高 | 需要建立 |
| 利润率 | 低（佣金15%+广告） | 高 |
| 品牌沉淀 | 弱 | 强 |
| 数据掌控 | 平台控制 | 完全掌控 |
| **当前选择** | 🟡 后期 | 🟢 当前优先 |

## 2.4 产品 Listing 优化 🟢【当前】

### 2.4.1 产品图片要求

| 图片类型 | 要求 | 示例 |
|---------|-----|------|
| **主图** | 白底、产品占比 85%、高清 | 产品正面 |
| **场景图** | 使用场景、生活方式 | 模特穿戴/使用中 |
| **细节图** | 材质、细节、功能点 | 特写镜头 |
| **尺寸图** | 标注尺寸、对比参照物 | 尺寸标注图 |
| **卖点图** | 图文结合展示卖点 | 信息图 |

### 2.4.2 产品文案结构（AI 可生成）

```
标题：核心关键词 + 品牌 + 主要卖点 + 规格
      例：Premium Yoga Pants | BrandX | High Waist Tummy Control | Black S-XL

五点描述：
• 核心卖点 1（解决什么问题）
• 核心卖点 2（材质/功能）
• 核心卖点 3（使用场景）
• 尺码/规格信息
• 售后保障（退换货政策）

详细描述：
品牌故事 + 产品详情 + 使用说明 + FAQ
```

### 2.4.3 关键词优化

| 关键词类型 | 说明 | 示例 |
|-----------|-----|------|
| 核心词 | 产品品类词 | yoga pants |
| 长尾词 | 更具体的搜索词 | high waist yoga pants for women |
| 品牌词 | 品牌名 | BrandX yoga pants |
| 竞品词 | 竞品品牌名（谨慎使用） | lululemon alternative |

## 3.1 达人网络

| 类型 | 粉丝量 | 合作成本 | 特点 |

| **小网红** | 1-10万 | $50-500 | 性价比高 |
| **中网红** | 10-50万 | $500-2000 | 有一定影响力 |
| **大网红** | 50万+ | $2000+ | 曝光量大 |
