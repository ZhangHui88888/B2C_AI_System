# 2.2 首页和布局

## 功能概述

实现响应式布局组件和首页各功能区域。

## 布局组件

### Header ([`Header.astro`](../../frontend/src/components/layout/Header.astro))

**功能**:
- 响应式导航栏
- 品牌Logo
- 主导航菜单（根据分类动态生成）
- 搜索框
- 购物车图标（带数量徽章）
- 移动端汉堡菜单

**响应式设计**:
- 桌面端：横向导航
- 移动端：汉堡菜单 + 侧边栏

**交互功能**:
- 购物车徽章实时更新
- 搜索框自动完成（可选）
- 下拉菜单（多级分类）

### Footer ([`Footer.astro`](../../frontend/src/components/layout/Footer.astro))

**内容区域**:
1. **链接分组**
   - 关于我们
   - 客户服务
   - 政策条款
   - 联系方式

2. **社交媒体**
   - Facebook、Instagram、Twitter等
   - 添加rel="noopener"安全属性

3. **支付方式**
   - Visa、Mastercard、PayPal等图标
   - SSL安全标识

4. **版权信息**
   - 公司名称
   - 年份
   - 备案信息（如需要）

## 首页区域

### Hero区域

**组成**:
- 大图/视频背景
- 主标题和副标题
- CTA按钮（立即购买/了解更多）
- 可选：轮播多个Hero

**设计要点**:
- 视觉冲击力强
- 文字清晰可读
- CTA按钮突出

### 热销产品轮播

**功能**:
- 展示Top 8热销产品
- 左右滑动切换
- 响应式网格（1/2/3/4列）
- 使用ProductCard组件

**性能优化**:
- 虚拟滚动（只渲染可见项）
- 图片懒加载
- 预加载前3个产品图片

### 分类展示网格

**布局**:
- 响应式网格（2/3/4列）
- 每个分类卡片包含：
  - 分类图片
  - 分类名称
  - 产品数量
  - 链接到分类页

**使用组件**: [`CategoryCard.astro`](../../frontend/src/components/catalog/CategoryCard.astro)

### 社会证明区域

**内容**:
- 客户评价轮播
- 信任徽章（支付安全、退货保证等）
- 媒体报道
- 销售数据（已服务XX客户）

### Email订阅表单

**功能**:
- 邮箱输入框
- 订阅按钮
- 隐私政策链接
- 成功/错误提示

**集成**:
- 连接邮件营销服务（Mailchimp/Klaviyo等）
- 存储到customers表

## 通用组件

### ProductCard ([`ProductCard.astro`](../../frontend/src/components/catalog/ProductCard.astro))

**显示内容**:
- 产品图片（hover效果）
- 产品名称
- 价格（原价/促销价）
- 评分星级
- 快速加购按钮

**交互**:
- 点击跳转详情页
- 快速加购弹窗
- 收藏按钮（可选）

### CategoryCard ([`CategoryCard.astro`](../../frontend/src/components/catalog/CategoryCard.astro))

**显示内容**:
- 分类图片
- 分类名称
- 产品数量
- 查看更多链接

**样式**:
- 卡片悬停效果
- 图片遮罩层
- 响应式布局

## 响应式设计

### 断点设置
```css
/* Mobile */
@media (max-width: 640px) { }

/* Tablet */
@media (min-width: 641px) and (max-width: 1024px) { }

/* Desktop */
@media (min-width: 1025px) { }
```

### 移动端优化
- 触摸友好的按钮尺寸
- 简化导航结构
- 优化图片加载
- 减少动画效果

## 性能优化

### 图片优化
- 使用WebP格式
- 提供srcset响应式图片
- 懒加载非首屏图片
- CDN加速

### 代码优化
- CSS按需加载
- JavaScript延迟加载
- 避免首屏阻塞
- 使用Intersection Observer

### 缓存策略
- 导航数据localStorage缓存
- 静态资源CDN缓存
- API响应缓存

## 可访问性

### ARIA标签
- 导航使用nav标签
- 按钮添加aria-label
- 图片添加alt属性

### 键盘导航
- Tab键顺序合理
- Enter/Space触发按钮
- Esc关闭弹窗
