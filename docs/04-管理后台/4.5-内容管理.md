# 4.5 内容管理

## 功能概述

管理博客内容、作者、评价和信任元素。

## 页面

### 内容管理 ([`/admin/content/index.astro`](../../frontend/src/pages/admin/content/index.astro))
- 博客文章列表
- 创建/编辑文章
- 发布/草稿管理
- SEO设置

### 作者管理 ([`/admin/authors/index.astro`](../../frontend/src/pages/admin/authors/index.astro))
- 作者列表
- 添加/编辑作者
- 作者信息管理
- 社交链接配置

### 评价管理 ([`/admin/reviews/index.astro`](../../frontend/src/pages/admin/reviews/index.astro))
- 评价列表
- 审核评价
- 回复评价
- 删除不当评价

### 信任元素 ([`/admin/trust/index.astro`](../../frontend/src/pages/admin/trust/index.astro))
- 媒体报道管理
- 认证标识配置
- 保障承诺设置

## 待开发功能

- 评价审核工作流
- 评价回复功能
- 内容版本控制

✅ **已完成** - 基础内容管理已实现
