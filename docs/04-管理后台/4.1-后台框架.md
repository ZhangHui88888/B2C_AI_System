# 4.1 后台框架

## 功能概述

搭建管理后台的基础框架，包括布局、认证和仪表盘。

## 核心组件

### AdminLayout ([`AdminLayout.astro`](../../frontend/src/layouts/AdminLayout.astro))

**布局结构**:
- 侧边栏导航
- 顶部栏（品牌切换、用户菜单）
- 主内容区
- 面包屑导航

### 登录页面 ([`/admin/login.astro`](../../frontend/src/pages/admin/login.astro))

**功能**:
- 邮箱密码登录
- Supabase Auth集成
- 登录状态保持
- 重定向到仪表盘

### 仪表盘 ([`/admin/index.astro`](../../frontend/src/pages/admin/index.astro))

**统计卡片**:
- 今日销售额
- 订单数量
- 访客数
- 转化率

**图表**:
- 销售趋势（7天/30天）
- 产品排行
- 分类分布

**快捷操作**:
- 待处理订单
- 待审核评价
- 库存预警

## 权限控制

### 角色定义
- super_admin - 超级管理员
- brand_admin - 品牌管理员

### 权限检查
- 前端UI控制
- 后端API验证
- RLS策略保护
