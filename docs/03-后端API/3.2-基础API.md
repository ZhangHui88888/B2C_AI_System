# 3.2 基础 API

## 功能概述

提供产品、分类、内容等核心数据的API接口。

## API端点

### 产品API ([`products.ts`](../../worker/src/routes/products.ts))

**POST /api/products/list** - 产品列表
- 支持分页、筛选、排序
- 按brand_id自动过滤

**GET /api/products/:slug** - 产品详情
- 返回完整产品信息
- 包含相关产品推荐

### 分类API ([`categories.ts`](../../worker/src/routes/categories.ts))

**GET /api/categories** - 分类列表
- 返回层级结构
- 包含产品数量

### 内容API ([`content.ts`](../../worker/src/routes/content.ts))

**GET /api/content/blog** - 博客列表
**GET /api/content/blog/:slug** - 博客详情

### 作者API ([`authors.ts`](../../worker/src/routes/authors.ts))

**GET /api/authors** - 作者列表
**GET /api/authors/:slug** - 作者详情
